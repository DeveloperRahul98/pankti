# Pankti — Documentation

This folder contains everything you need to understand the Pankti project, regardless of your background.

| Document | Who it's for | What's inside |
|---|---|---|
| [PRD.md](./PRD.md) | Business owners, product folks, new team members | What the product is, who it's for, the problem it solves, every feature, business goals, and the roadmap |
| [TECH-STACK.md](./TECH-STACK.md) | Anyone curious about how it's built — including non-engineers | Plain-English explanations of every technology used, the folder layout, and the deliberate things we chose *not* to use |
| [USER-GUIDE.md](./USER-GUIDE.md) | Customers, family, anyone using the site for the first time | Every feature explained as a tour — how to build a plate, send an enquiry, use the gallery viewer, etc. Includes a sample user journey at the end |

All three are written so someone with no technical background can read them top to bottom and come away understanding the project.
